Maintenance Guide

How to update your website

This guide tells you exactly how to change anything on this site — no coding experience needed. Read the relevant section before making any changes. Always keep a backup copy of the file before editing.

Before you change anything: Make a copy of the HTML file and save it somewhere safe. If something breaks, you can always go back to the copy. Rename it something like monicah-portfolio-BACKUP.html.
📹 How to add your intro video
1
Record your video

Record a 60–90 second video using your phone or laptop camera. Use the script on the Home page. Good lighting matters more than a fancy camera.

2
Upload to YouTube or Loom

Upload to YouTube (set to "Unlisted" so only people with the link can see it) or Loom (free plan works fine). Copy the embed link.

3
Find this section in the HTML file
Search for this text (Ctrl+F):
<div class="vid-ph">

Replace the entire vid-ph div with your YouTube/Loom embed code. Example:

<iframe width="100%" height="100%" src="https://www.youtube.com/embed/YOUR_VIDEO_ID" frameborder="0" allowfullscreen></iframe>
📄 How to add your resume PDF download
1
Upload your PDF to Google Drive

Upload your resume PDF to Google Drive. Right-click → Share → "Anyone with the link can view". Copy the link.

2
Find this text in the file
Search for:
YOUR_RESUME_PDF_LINK_HERE

Replace it with your Google Drive link. It should look like: https://drive.google.com/file/d/XXXX/view

🏅 How to add certificate images or links
1
Get your certificate link from Coursera/Credly

Go to your certificate on Coursera or Credly. Copy the shareable URL (it usually looks like coursera.org/verify/XXXX).

2
Find the certificate card in the HTML
Search for:
Add your cert link here

There are 5 of these — one per certificate. Replace the text with a clickable link:

<a href="YOUR_CERT_URL" target="_blank" style="font-size:0.6rem;color:var(--ink)">View certificate →</a>
✏️ How to change any text
1
Open the file in a text editor

Right-click the HTML file → Open with → Notepad (Windows) or TextEdit (Mac). Or use VS Code if you have it — it's free and much easier to read.

2
Use Find (Ctrl+F or Cmd+F)

Search for the exact words you want to change. This is faster than scrolling. Find the text, change it, save the file.

3
Save and re-upload

After saving, re-upload the file to Tiiny.host or Netlify. Your link stays the same — the content just updates.

📊 How to add a new case study
1
Find the case studies section
Search for:
<!-- CASE 4 -->

Copy the entire block from <!-- CASE 4 --> down to the next <!-- CASE comment. Paste it after Case 4.

2
Update the content

Change the tag color (cs-tag-g for terracotta, cs-tag-a for amber), the title, the challenge description, the "What I did" bullet points, and the results pills.

🌐 How to host and share the site (free)
1
Tiiny.host (easiest)

Go to tiiny.host. Drag and drop your HTML file. You'll get a link like monicah.tiiny.site. Free plan works fine. To update: just re-upload the file with the same name.

2
Netlify Drop (also free, slightly more stable)

Go to app.netlify.com/drop. Drag your HTML file onto the page. You'll get a free URL. You can also connect it to a custom domain later if you buy one.

3
Custom domain (optional, ~$10/year)

Buy a domain at Namecheap or Google Domains — something like monicahmburu.com. Connect it to Netlify for free. This makes your link look fully professional.

🔒 How to hide the Guide page from visitors
1
The Guide tab is hidden from the main nav

Only you know the URL. To access it, go to your site URL and add #/guide at the end — like monicah.tiiny.site#/guide. Clients will never see it unless you send them that link.
